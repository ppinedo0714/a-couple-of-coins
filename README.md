# a-couple-of-coins

A money tracking and budget app with predictive analytics. Built as a monorepo with a React frontend, Go REST API, and Python prediction service.

## Services

- [`frontend/`](./frontend/CLAUDE.md) — React + TypeScript UI
- [`backend/`](./backend/CLAUDE.md) — Go REST API
- [`services/predictions/`](./services/predictions/CLAUDE.md) — Python FastAPI service for predictive budget analytics

Setup instructions will be added as each service is scaffolded. See [`CLAUDE.md`](./CLAUDE.md) for the architecture overview and per-service docs.
