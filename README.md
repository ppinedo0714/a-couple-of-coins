# a-couple-of-coins

A money tracking and budget app with predictive analytics. Built as a monorepo with a React frontend, Node.js REST API, and Python prediction service.

## Services

- `frontend/` — React + TypeScript UI
- `backend/` — Node.js + Express REST API
- `services/predictions/` — Python FastAPI service for predictive budget analytics
